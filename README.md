## Nombre del proyecto

GGamer

## Descripcion

GGamer es una aplicacion web estatica de tipo e-commerce orientada a la venta de productos gamer. Incluye una pagina principal, categorias de productos, formularios de login y registro, y un diseno visual con estilo gamer para la primera etapa del proyecto.

## Estilos

Los estilos fuente se encuentran en `scss/styles.scss` y el sitio utiliza el CSS compilado en `css/styles.css`.

Comandos disponibles:

```bash
npm install
npm run build:css
npm run watch:css
```
